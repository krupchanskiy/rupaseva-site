Vedarama

We have created a convenient shell for a large electronic Vaishnava library, which has been collected by Srila Bhakti Vijnana Goswami for many years.

The main functions:

  • Convenient catalog of books in Russian, English and Sanskrit with the ability to quickly search for books.
  • Search engine with different algorithms for issuing results.
  • Ability to quickly find different commentaries on the same text.
  • Convenient search using word-by-word translations, which allows you to find out different meanings of a word.
  • A separate block with dictionaries and glossaries (only in English). It contains all the main Sanskrit dictionaries.
  • Converter for translating text with diacritics from one encoding to another. You can also translate text with diacritics from Russian to English and vice versa.
  • The library works without an Internet connection. The book database is updated online.

Access levels

We provide basic access to everyone, which includes all of Srila Prabhupada's books, the major acarya commentaries on the Bhagavad-gita and Srimad-Bhagavatam, and some academic publications. Full access to all books is limited to authorized ISKCON researchers.

Win version
